BQN-80


Write a function that gets timestep (starting from 0, incrementing on each frame) as left argument and previous frame as right argument, and returns a frame (240x136 array of numbers, which will be mapped to a 16-color palette with & 0x0f). Note that BQN-80 is still experimental and everything may still change.


Standard BQN keymap works with the prefix \.
